Setup Server Frontend with ReactJS.

01- Create files /cs-client and /cs-server.
02- Open terminal and change directory to /cs-client.
03- Type in terminal -> git clone https://github.com/dmjau/cs-em-client.git
03- Type in terminal -> npm i (install all dependencies on package.json)
04- Type in terminal -> yarn start (Runs the app in the development mode).
